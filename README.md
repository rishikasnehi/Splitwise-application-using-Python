# Splitwise Application Using Python

A simple command-line tool to help groups split expenses, track balances, and simplify repayments‚Äîjust like the Splitwise app but built using Python.

## üöÄ Why This Project Is Useful

- **Automates Expense Splitting:** Quickly divide payments among group members and keep a running total of who owes what.
- **Tracks Balances:** Displays outstanding balances, letting everyone know if they owe or get back money.
- **Debt Simplification:** Calculates simplified repayments so fewer transactions are needed to settle up.
- **Easy to Use:** Interactive prompts guide users through adding members, recording expenses, and viewing balances.

### Prerequisites

- Python 3.x installed on your system.

### Installation

Clone the repository:
```sh
git clone https://github.com/rishikasnehi/Splitwise-application-using-Python.git
cd Splitwise-application-using-Python
```

### Usage

Run the application:
```sh
python SplitWise.py
```

**Features you can choose:**
1. Add Members: Set up the names in your group.
2. Add Expense: Enter details (payer, amount, participants, optional note) for each payment.
3. Show Outstanding Balance: See who owes or gets back money.
4. Simplify Debts: Get an easy-to-follow summary for paybacks.
5. Exit: Leave the app when you‚Äôre done.
